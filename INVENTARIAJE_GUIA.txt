COMIENZA EN 30 MINUTOS

1. LEE ESTO PRIMERO (2 min)
Lee el archivo RESUMEN.md. Es muy simple y rapido.

2. BACKEND EN VERCEL (10 min)
- Ir a vercel.com
- Conectar GitHub
- Crear nuevo proyecto
- Subir archivos
- Deploy automatico
- Copiar URL: https://tu-proyecto.vercel.app

3. PREPARAR APP (10 min)
- Instalar Node.js
- npm install -g expo-cli
- mkdir InventariAJE
- cd InventariAJE
- Copiar todos los .jsx y package.json
- npm install
- Actualizar API_URL en todos los archivos
- npm start

4. PROBAR EN TELEFONO (5 min)
- Descargar "Expo Go"
- Escanear codigo QR
- Login: admin

5. COMPILAR APK (10 min)
- npm install -g eas-cli
- eas login
- eas build --platform android --local
